# 🌐 Afzal Momin — Personal Portfolio

A fully responsive personal portfolio website built from scratch using HTML, CSS, and JavaScript — featuring particle animations, flip project cards, radial skill charts, and smooth scroll interactions.

🔗 **Live:** [afzalmomin-6202.github.io/My-Portfolio](https://afzalmomin-6202.github.io/My-Portfolio)

---

## ✨ Features

- 🎇 Interactive particle animations on every section using **particles.js**
- ⌨️ Typing animation in the hero section using **Typed.js**
- 🃏 Flip cards for project showcase
- 📊 Animated radial skill charts for professional skills
- 📈 Animated progress bars for technical skills
- 📬 Contact form that opens Gmail with pre-filled message
- 📱 Fully responsive design for all screen sizes
- 🎓 Education timeline with horizontal scroll

---

## 🗂️ Sections

| Section | Description |
|---|---|
| **Home** | Hero section with name, typing animation, and social links |
| **About** | Brief introduction and downloadable CV |
| **Skills** | Technical skills (progress bars) and Professional skills (radial charts) |
| **Projects** | Flip cards showcasing 4 projects with GitHub links |
| **Education** | Timeline of academic qualifications |
| **Contact** | Contact form + email, phone, location details |

## 🛠️ Technologies Used

- **HTML5** — Structure and semantic markup
- **CSS3** — Styling, animations, flip cards, responsive design
- **JavaScript** — Interactivity, form handling, scroll effects
- **particles.js** — Animated particle backgrounds per section
- **Typed.js** — Typewriter effect in hero section
- **Boxicons** — Icons throughout the site
